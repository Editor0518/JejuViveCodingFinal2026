# 인명용한자 보강 데이터 — 설명서

구글 스프레드시트 백엔드용으로 정리·보강한 파일입니다.

## 파일
- `인명용한자_현행전체목록_보강.csv` — 보강 메인 데이터 (10,444행, UTF-8, BOM 없음)
- `인명용한자_개정연혁_정리.csv` — 개정연혁 (BOM만 제거)
- 원본 2종은 그대로 보존

## 데이터 출처
- 획수·부수·영문뜻: KANJIDIC2 (jamdict, 오프라인)
- 발음오행·초성·유니코드·id: 원본에서 계산

## 주의사항 (중요)
1. **미할당 유니코드 442자** — 원본에 U+A0xxx 등 유니코드 **미할당 영역** 코드가 442자 있습니다. 이는 대법원 전산표의 임시 내부코드로 추정되며, 표준 폰트에서 렌더링·검색이 되지 않습니다. `유니코드상태=미할당`으로 표시했으니, 정식 유니코드/이체자로 치환하거나 별도 처리 필요.
2. **획수 기준 차이** — `총획수`는 KANJIDIC2(일본식) 기준이라 일부 글자는 성명학의 강희자전 원획과 다를 수 있습니다(예: 龜 = 18 vs 한국 관행 16). 수리작명에 쓰려면 한국어문회/강희자전 원획으로 교차검증 권장.
3. **훈(한글 뜻) 미포함** — 한글 훈("밝을 명" 등)은 오프라인 확보가 어려워 미포함. 한국어문회 등급별 한자셋(강희 원획+한글 훈+급수 포함)을 확보해 `id` 키로 병합하면 완성됩니다. (허용자체표와 동일하게 추후 병합 가능)
