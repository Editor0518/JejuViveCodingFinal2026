# 한자 작명 도우미 — 서비스 소개

## 한눈에 보기

작가·캐릭터 디자이너·창작자가 만든 캐릭터의 **한글 이름**에 어울리는 **한자(漢字)**를 찾아 이름의 뜻을 정하도록 돕는 단일 페이지 웹 도구(`index.html`)입니다. 실제 출생신고·개명에서 쓰이는 **인명용 한자** 기준을 그대로 반영해, 소리(한글)와 의미(한자)를 함께 고려하면서 제도상 통용되는 범위 안에서 설득력 있는 이름을 지을 수 있게 합니다.

별도 서버·빌드 없이 브라우저에서 바로 동작하며, 한자·성씨·개정연혁 데이터는 구글 스프레드시트(Apps Script 웹앱 JSON)에서 실시간으로 불러옵니다.

## 화면 구성

상단 네비게이션으로 네 개의 페이지를 오갑니다. `display` 토글 방식이라 페이지를 이동해도 입력값과 검색 결과가 사라지지 않습니다.

**소개** — 서비스의 목적과 제공 기능을 안내하는 페이지입니다.

**작명** — 핵심 기능이 모인 페이지로, 아래 두 탭으로 나뉩니다.

- *한글로 이름짓기*: 조건에 맞는 한글 글자를 찾아줍니다. 완성 글자·자음·모음을 섞어 입력하는 **간편 검색**과, 자모를 한 칸씩 넣고 화면 자판까지 지원하는 **심화 검색**이 있습니다.
- *한자 찾기*: 이름 글자별로 쓸 수 있는 인명용 한자와 뜻을 보여주고, 성씨로도 쓰이는 한자(姓)를 붉게 따로 표시합니다. '성씨만 보기', 성씨 글자 수 지정, '글자 완벽히 맞춤' 옵션을 제공합니다.

**도우미** — 이름 메모에서 선택한 후보를 바탕으로, 이름 한자의 뜻 풀이를 요청하는 **AI 프롬프트**를 자동 생성합니다. 생년월일·성별(선택 입력)을 함께 담아 복사할 수 있으며, 이 도구가 직접 AI에 묻지는 않고 사용자가 ChatGPT·Gemini·Claude 등에 붙여넣는 방식입니다. 네이버 한자사전, 대법원 전자가족관계등록시스템 등 관련 링크도 안내합니다.

**개정 연혁** — 인명용 한자 제도가 1991년 도입 이후 확대·정비되어 온 과정을 연도·시행일·추가 글자수·누적 글자수·근거법령별로 정리한 표입니다(2024년 개정본이 2026년 현재 유효).

## 특징적인 기능

**시행일 기준 검증** — 출생신고·개명 신청일을 지정하면, 시행일이 그날 이후여서 당시엔 인명용으로 쓸 수 없던 한자를 검색 결과에서 따로 구분해 보여줍니다(시행일 당일 신청은 사용 가능으로 처리). 창작물의 시대 배경에 맞는 고증에 유용합니다.

**한글 자모 엔진** — 완성 글자를 초성·중성·종성으로 분해하고 이중모음·겹받침을 원자 단위로 풀어, 초성만·모음만·부분 일치까지 유연하게 검색합니다.

**이름 메모 패널** — 오른쪽 고정(모바일은 하단 버튼) 패널에서 여러 이름 후보를 관리합니다. 검색 결과의 ☆로 "한글 한자(뜻)" 또는 한글 글자만(한자 미정)을 담고, 후보를 ●로 선택해 활성화하며, 캐릭터별 신청일 기록과 메모, 전체/개별 복사를 지원합니다.

**데이터 정제** — 스프레드시트에서 불러온 한자 중 미배정·PUA 등 표기가 깨지는 코드포인트는 자동으로 걸러냅니다.

**영타 → 한글 자동 변환** — 작명 탭에서 한/영 전환을 깜빡하고 영어 자판으로 입력하면(예: `rk` → `ㄱㅏ`), 대응하는 한글 자모로 즉시 바꿔줍니다. 지우고 한/영 키를 다시 누를 필요가 없습니다. 두벌식 표준 자판 기준이며, Shift 조합(쌍자음·ㅒ·ㅖ)도 반영합니다. 별도 권한 없이 감지 가능한 CapsLock 상태를 읽어, CapsLock이 켜져 대소문자가 뒤집힌 경우도 바로잡아 매핑합니다. 변환은 간편검색·한자찾기·심화검색 입력칸에서만 동작하며, **이름 메모 패널에서는 영타 변환을 하지 않아** 영문 메모를 그대로 적을 수 있습니다.

## 기술 구성

순수 HTML·CSS·JavaScript로 만든 단일 파일이며 외부 프레임워크를 쓰지 않습니다. 데이터는 코드 내 `SHEET_API_URL`(Apps Script 웹앱 주소)에서 `fetch`로 받아오고, 로드 상태는 상단 배너로 표시됩니다. 서체는 Noto Sans/Serif KR을 사용하며, 따뜻한 한지 톤의 색상과 반응형 레이아웃(1100px 기준으로 메모 패널 고정/토글 전환)을 갖췄습니다.
