# 인명용추가한자 데이터베이스 정리 노트 (1992–2007 텍스트 추출분)

## 산출물

- **`인명용추가한자_1992-2007.csv`** — 시행일별로 새로 추가된 한자 목록 (3,253자)
- **`처리현황_파일목록.csv`** — 폴더 내 전체 규칙 파일의 처리 상태(사용/부분/스킵) 목록

## CSV 열 구성

| 열 | 설명 |
|---|---|
| 한글 | 해당 한자가 인명용추가한자표에 등재된 독음(음) |
| 한자 | 한자 1자 |
| 시행일 | 해당 한자가 처음 등장한 개정 규칙의 시행일 |
| 대법원규칙호 | 해당 개정의 대법원규칙 호수 |
| 개정일 | 해당 규칙의 개정(공포)일 |
| 출처파일 | 추출에 사용한 원본 파일명 |
| 한글출처 | 독음의 근거 (아래 "독음 신뢰도" 참조) |

## 방법론

각 규칙의 별표1(인명용추가한자표)은 **그 시점까지 누적된 전체 추가한자**를 다시 싣는 구조다. 따라서 "그 개정에서 새로 추가된 한자"는 **연대순으로 앞 파일과의 차집합**으로 계산했다.

- 1992~1996년 파일: `[가] 嘉 嫁 …` 형식(대괄호 독음). 텍스트에서 독음-한자를 정확히 파싱.
- 1998~2007년 파일: 2단 표 형식. PDF의 **표 괘선(행·열 구분선)** 을 이용해 각 셀(독음칸+한자칸)을 매칭 → 독음을 정확히 배정.
- 한자 집합(날짜 판정용)은 각 파일에서 추출한 **전체 한자의 합집합**을 기준으로, 문자별 **최초 등장 개정**을 부여.

## 독음 신뢰도 (한글출처 열)

- `표(정자체)` (1,157자): 대괄호형 표에서 직접 파싱 — 정확.
- `표` (2,079자): 괘선 표 셀 매칭 + 여러 개정본 다수결 — 정확. 다수결이 강한 이차독음(예: 金→김, 茶→차, 北→배, 宅→댁)은 표기 그대로 유지.
- `표→사전보정` (15자): 표 파싱 근거가 약한 희귀자를 한자사전 독음으로 보정.
- `추정(사전)` (2자): 표에서 독음을 확보하지 못해 사전 독음으로 채움.

## 한계 및 주의

1. **이미지 파일 미포함(스킵)**: 2008년 이후 규칙은 한자가 이미지(HWP·PDF·HWPX)로 되어 있어 OCR이 필요하다. 이번 정리에서는 제외했으며, 전체 목록은 `처리현황_파일목록.csv`에 정리했다. (1991-04-01 파일은 별표2 인명용한자허용자체표로 별도 대상.)
2. **부분추출 파일(2004-03-31, 2005-10-24, 2006-05-25)**: 일부 페이지가 이미지라 완전 추출이 안 된다. 이 구간의 추가분은 인접한 완전 추출 개정본으로 통합 반영했다. 특히 제1959호(2005-10-24)의 개정 고유 추가분은 신뢰도 있게 추출되지 않아 별도 등재하지 않았다(각주·기초한자 유입분은 제외 처리).
3. 각주("첫소리(初聲)…")·예시에서 유입된 비표제자(初, 聲, 女, 山, 王, 爭)는 제외했다.

## 2008–2011 이미지 한자 OCR 반영 (2026-07-15)

`변환결과_markdown/이미지한자_OCR결과.csv`(개별 글리프)와 `…_음별펼침.csv`(음별 3,248쌍)의 OCR 결과를 본 데이터베이스에 반영했다. **완전 일치(동일 음+동일 코드포인트) 중복만 제외하고 전부 반영**하는 방침을 적용했다.

- **개별 글리프(제2119·2181·2188·2227·2242호, 특수한자)**: 전량 `인명용한자_현행전체목록.csv`에 이미 등재되어 있어 신규 추가 없음(등재 확인).
- **BMP 클러스터(제2299·2372호)**: 음별 3,248쌍 추출.
  - `인명용한자_현행전체목록.csv`(현행 마스터, 3열 스키마 유지)에 **신규 97행 추가**(10,347→10,444행). 이 중 138건은 인명용한자표 원형인 **호환한자(U+F900 블록) 이차독음**(예: 金→김, 北→배, 茶→차), 18건은 순수 신규 한자.
  - 이력DB를 **`인명용추가한자_1992-2011.csv`로 확장**(3,253→3,397행). OCR 유래 144행은 `한글출처`를 `이미지OCR`(각주 오염 의심 4건은 `이미지OCR-주의(각주오염가능)`)로 표기하고, 시행일은 **최초 관측 규칙** 기준으로 부여했다.
- **주의**: 정리노트가 지목한 각주 오염 비표제자(女·山·王·爭 등) 4건은 방침상 포함하되 `이미지OCR-주의` 및 변경내역의 신뢰도 플래그로 구분했다. 필요 시 제거 권장.
- **산출물**: `변환결과_markdown/OCR반영_변경내역.csv`(반영 162쌍 상세·플래그), 원본은 `*.csv.bak` 백업 보존.
- **처리현황**: `처리현황_파일목록.csv`·`변환결과_처리내역.csv`에서 제2119~2372호 항목을 `OCR반영완료`로 갱신. **제3151·3140·3220호(2024~2025, 7,274 글리프)는 미처리(OCR 대기)로 남음.**
